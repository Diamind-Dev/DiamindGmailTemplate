I'm happy to confirm that ANFP should be well-suited for your case. You can use ANFP to receive file paths for importing 3D models from both local storage and Google Drive.

Here’s the suggested workflow to verify everything works smoothly:

1.Use the ANFP Demo Edition to select an .ifc file from either local storage or Google Drive.
2.Pass the returned path directly to the "Assimp Import Files" node from the UE-Assimp plugin.
3.If the import works as expected, you can complete your pipeline using:

The "File Picking Result" event dispatcher for local storage, and
The "On Download Success" event dispatcher for Google Drive (both available in ANFP).
